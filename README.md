# Data Structures Basics in C Language

Welcome to the **Data-Structures-Basics-in-C-langauge** repository! This repository serves as a comprehensive, structured log of fundamental data structure implementations and algorithmic problem-solving using the C programming language. 

The codebase covers core curriculum concepts ranging from basic space/time complexity analysis to advanced structures like Trees, Hashing, and Indexing.

---

## 📂 Repository Organization & Syllabus Mapping

The repository consists of modular laboratory folders (`Lab_01` through `Lab_14`) and specialized directories. Below is the mapping of how these folders align with core Data Structure concepts:

### 🔬 Core Concepts Covered
* **Asymptotic Analysis & Algorithm Performance**
    * Frequency Count Method (Time & Space Complexity)
    * Practice problems on Big-O, Theta ($\Theta$), and Omega ($\Omega$) notations
* **Linear Data Structures**
    * **Arrays:** Basic operations (Insertion, Deletion, Traversal), Multi-dimensional arrays
    * **Stacks:** Infix to Postfix conversion logic, Evaluation of expressions
    * **Queues:** Linear Queue, Circular Queue implementation
    * **Linked Lists:** Singly, Doubly, and Circular Linked Lists with standard algorithmic operations
* **Non-Linear Data Structures**
    * **Trees & Binary Search Trees (BST):** Algebraic expression representation into Binary Trees, Tree Traversals (In-order, Pre-order, Post-order)
    * **Advanced Trees:** B-Trees and Balanced Trees
* **Searching, Hashing & Indexing Techniques**
    * Hash functions, collision resolution mechanisms
    * Indexing files for optimized data retrieval
